This is a peer-graded assignment which is part of the "Getting and Cleaning Data" course offered by the Johns Hopkins University via Coursera.
R version 4.0.5 (2021-03-31) -- "Shake and Throw" was used to complete this assignment.
The submission for this assignment contains an R script which produces two tidy data sets, a code book containing information about the study design and variables, and a README file.

R script

--Step 0-- 
Creates subdirectory for the "untidy" data to be placed into upon downloading, downloads data files in Zip-format from URL, and unzips data files for them to be made accessible.

--Step 1--
Merges the training and the test sets to create one data set.

--Step 2--
Extracts only the measurements on the mean and standard deviation for each measurement.

--Step 3--
Uses descriptive names to name the activities in the data set.

--Step 4--
Appropriately labels the data set with descriptive variable names.

--Step 5--
Creates a second, independent tidy data set with the average of each variable for each activity and each subject.
